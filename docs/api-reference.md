# API reference

This repository does not duplicate full endpoint schemas in Markdown. The **source of truth** is the **OpenAPI** schema generated by FastAPI at runtime.

## Swagger UI

With the API running (see [Getting started](./getting-started)):

| Scenario | URL |
|----------|-----|
| Default local | `http://localhost:8000/docs` |
| With `BASE_PATH` set (e.g. `/agentic_web_starter`) | `http://localhost:8000{BASE_PATH}/docs` |

Use **Try it out** to call endpoints with your JWT and request bodies.

## ReDoc

ReDoc is usually available at **`/redoc`** (same base URL rules as above).

## Core vs IAM

- **Core** — e.g. health, chat, upload (subject to `USE_QDRANT`, RabbitMQ, etc.).
- **IAM** — auth and user/role CRUD only when **`USE_SQL`** is enabled.

If a route does not appear in OpenAPI, it was not registered (check feature flags and `app/starter.py`).
